### 01. CommonJS 방식:
  - `.js` 확장자 사용
  - `require()`와 `module.exports` 사용
  - 추가 설정 없이 바로 사용 가능

### 02. ES Modules 방식 (`.mjs` 파일 사용):
  - `.mjs` 확장자 사용
  - `import`와 `export` 사용
  - Node.js에서 바로 실행 가능

### 03. ES Modules 방식 (`.js` 파일 + package.json):
  - `.js` 확장자 사용
  - `import`와 `export` 사용
  - `package.json`에 `"type": "module"` 설정 필요
